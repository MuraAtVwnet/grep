grep by PowerShell

■ これは何?
いわゆる grep の PowerShell 実装

■ 使い方
grep 正規表現 ファイルパターン -Recurse

■ 例
grep 18[0-9] c:\temp\*.txt -Recurse

■ Install 方法
install.ps1 を実行してください

■ Uninstall 方法
uninstall.ps1 を実行して下さい
(問い合わせが来たら Enter)

■ 動作確認環境
Windows PowerShell 5.1
PowerShell 7.1.3(Windows)
